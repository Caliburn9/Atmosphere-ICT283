Returns

Precondition
The input stream must be formatted with comma separated values (e.g. "ID,Name,Credits"), and the value of Credits must be a valid number data type.
Postcondition
The relevant Unit data will be extracted from the input stream and stored in the passed Unit object, and the input stream will be advanced passed the consumed data.
