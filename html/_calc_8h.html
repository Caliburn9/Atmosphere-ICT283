Calculates the sample standard deviation of a vector of values.

This templated function uses the sample standard deviation formula, where the variance is divided by (N - 1). It assumes a precalculated mean is provided.

Template Parameters
T- The type of the elements stored in the vector (e.g., float, int).
Parameters
vec- A vector containing elements of type T.
N- The number of elements to include in the standard deviation calculation.
mean- The precalculated mean of the vector elements.
Returns
The sample standard deviation of the elements in the vector.
Precondition
N > 1 and all elements in vec must be valid numeric values.
Postcondition
Returns the sample standard deviation of type T.
