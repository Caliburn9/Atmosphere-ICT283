Merges two sorted halves of a vector into a single sorted segment.

This function merges two sorted sub-vectors (from start to mid and mid+1 to end) into a single sorted segment in the original vector.

Template Parameters
TThe type of elements in the vector.
Parameters
dataThe original vector containing the segments to merge.
startThe starting index of the first sorted sub-vector.
midThe ending index of the first sorted sub-vector (mid + 1 is the start of the second).
endThe ending index of the second sorted sub-vector.
Precondition
data[start...mid] and data[mid+1...end] must be individually sorted.
Postcondition
data[start...end] is fully sorted and duplicates are retained.

Sorts a vector using the Merge Sort algorithm.

This recursive function implements the Merge Sort algorithm to sort elements in a vector between the given start and end indices.

Template Parameters
TThe type of elements in the vector.
Parameters
dataThe vector to be sorted.
startThe starting index of the segment to sort.
endThe ending index of the segment to sort.
Precondition
data must be a valid Vector<T> with elements at indices from start to end.
Postcondition
The segment data[start...end] is sorted in ascending order.
