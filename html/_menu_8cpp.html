void CalculateAndDisplaySPCC ( const BST< AtmosRecType > & data)

Calculates and displays Sample Pearson Correlation Coefficients for a specified month.

This function prompts the user for a target month and calculates three Sample Pearson Correlation Coefficients:

  • Wind Speed vs Air Temperature (S_T)
  • Wind Speed vs Solar Radiation (S_R)
  • Air Temperature vs Solar Radiation (T_R)

It uses the static Collector class to gather filtered data from the provided BST of atmospheric records. Data is collected via in-order traversal using function pointers to Collector methods. Each correlation coefficient is then computed using the sPCC methods and printed to the console.

Parameters
dataA constant reference to a BST of AtmosRecType records, representing all atmospheric data.
Returns
void
Precondition
The BST must be populated with valid AtmosRecType data. The Collector class must be correctly implemented.
Postcondition
No changes to the BST. Output is printed to standard output. Collector’s internal state is reset between each correlation.
