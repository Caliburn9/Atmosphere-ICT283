Recursively builds a height-balanced Binary Search Tree from sorted atmospheric data.

This function uses a recursive divide-and-conquer strategy to construct a balanced BST by inserting the middle element of the current range, then repeating the process for the left and right subranges.

Parameters
bstDataA reference to the BST where the balanced nodes will be inserted.
sortedDataA const reference to a sorted AtmosLogType (vector of AtmosRecType).
startThe starting index of the current range in the sorted vector.
endThe ending index of the current range in the sorted vector.
Returns
void
Precondition
sortedData must be sorted by the desired BST ordering criteria (e.g., Date/Time).
Postcondition
bstData is populated with a balanced BST containing all elements from sortedData[start] to sortedData[end].

bool LoadAtmosphereData ( AtmosLogType & atmosData)

Loads atmospheric data from multiple input files listed in data_source.txt.

This function opens the "data/data_source.txt" file and reads filenames line by line. For each listed file, it attempts to open and parse atmospheric records using ReadAtmosphereData(). All valid data is appended to the provided AtmosLogType vector.

Parameters
atmosDataA reference to an AtmosLogType (i.e., Vector of AtmosRecType) where all parsed records are stored.
Returns
true if data_source.txt was successfully opened and at least one input file was attempted, and false if data_source.txt could not be opened.
Precondition
data/data_source.txt must exist and contain valid input filenames, one per line.
Postcondition
atmosData contains all valid parsed records from the listed files. Invalid files are skipped with a warning.
